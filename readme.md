# 题目7：语音超声波测距尺
## 要求：（此项目如果没有语音播报功能算零分）
1. 测距功能：利用超声传感器通过时间差测距法测距，将电信号送入单片机，利用数码管显示超声波与障碍物距离  
>   探测距离：2cm-450cm，精度：0.5cm  
    距离显示准确，稳定  
    便携式系统，USB供电  
2. 语音播报功能：将距离设为三档进行语音提示。  
>   一档：当障碍物大于等于2米时，语音：大于两米，只说一遍  
    二挡：当障碍物大于1米小于两米时，语音：大于一米，每隔5秒说一次，重复说  
    三档：当障碍物小于1米时，语音：小于一米，每隔1秒说一次，重复说。  
3. C语言编程  
>   硬件：用Altium designer画图，在实验室自制电路板（热转印法）  
    超声测距传感器、语音系统、STC89C52RC，ISP下载线、必要的运放及电阻、电容等元件。  
    软件：Altium designer软件，编程软件，程序烧写软件等。
